一、股癌 EP671 核心論點(原始前提)

💡 外溢效應:AI 伺服器與 800V 高壓爆發時,海外 IDM(英飛凌、TI)把毛利 40–50%+ 的高階品(DrMOS / 高階 SiC)留著,中低階訂單「外溢」給台廠。
💡 地緣紅利:Nexperia(安世)受制裁 → 大廠「非中化」提早布局,台系成二線供應商驗證對象。
💡 800V DC 破冰:即使明年滲透率僅 5%,GB300 / Rubin 單顆 1800W 也讓功率元件「用量 + 密度」大增。
💡 先行指標:觀察英飛凌、安森美等美股 IDM 巨頭動向,確認趨勢啟動。

★ 本報告「相信上述論點正確」,並以當前數據檢驗:論點若成立,錢會怎麼流、哪一段彈性最大、現價是否已反映。

二、股癌 EP672 新增 / 更新論點(本次校準 · 假設內容皆正確)

EP672 並未推翻 EP671,而是把外溢效應的「強度、時間軸與贏家輪廓」講得更清楚。以下五點是本版調整論述的依據:

⏳ 限速器理論
消費性電子與車用需求疲弱(「鳥」),在分析表上是負項,對 AI 功率循環卻是好事:像限速器一樣防止需求一次爆衝→價格崩壞(需求破壞),讓這波硬體漲價循環走得更久、更長。
→ 循環仍在「幼年期」,即便中途 −30% 修正,長期向上
📛 Lead Time「99+」
歐美一線 IDM 高階功率料件交期已拉到一年以上,甚至掛出 99+(根本排不到交期)。聽眾(工廠端)回饋:功率料件缺貨程度比被動元件更嚴重。
→ 可量化的供需吃緊指標,放進報告當佐證
🔁 二工認證機會大增
IDM 產能卡死 → 客戶被迫尋找 second-source(二工)。台廠「近水樓台先得月」,在高壓 MOSFET / 伺服器 MOSFET 的認證與一起出貨機會大幅增加。
→ 上調台灣 IDM(強茂/台半)與架線股的訂單動能
🏭 產能即王道
漲價循環中,握有產能(IDM)或能 secure 到代工產能(Fabless)的公司才是贏家;拿不到產能,有訂單也出不了貨。股癌:已注意到六/八吋代工廠股價相對走強(市場提前卡位)。
→ 上調代工產能權重(個股對應為本報告推論:世界先進 / 漢磊)
📊 Vishay 觀察指標
Vishay 同時有被動 + 功率業務;複盤發現其功率半導體 Lead Time 升幅遠比被動更兇,是證明「功率比被動更吃緊」的關鍵盤面指標。
→ 新增一個外部交叉驗證的觀察點
💧 資金輪動 + 集體表態
資金從被動元件(國巨 / 高階 MLCC / 電感)輪動到功率半導體(IC / MOSFET);美系、日系、中系功率股(如士蘭微)同步集體表態 → 全球性輪動,真實性較高(非單股人為噴出)。
→ 籌碼面驗證趨勢確立
⚠️ EP672 風險提醒:①需求破壞——價格不可能無限上漲,漲到消費端撐不住(如 iPhone 漲價反應)會鎖死需求; ②單股噴出——若非族群集體表態而是單一大噴,警惕人為操作;③拿不到產能的小 Fabless——有單也出不了貨,吸引力應調降; ④不要預測「第幾局」——隨機應變比預言重要,依盤面指標拆招。

★ EP672 未談個股 forward PE / 晨星公允價值倍數,亦未逐一細講強茂、台半、順德等台股代號(以「台系 MOSFET 業者 / 二工廠商 / 族群輪動」概念帶過); 本版財務數據沿用原報告(Futu + FinMind),EP672 僅在質化論述、時間軸與評分權重上校準。

三、三段價值鏈 · 分工與金流方向

① 海外 IDM 英飛凌 / 安森美 / 意法 / TI / 羅姆  — 99+ 交期外溢中低階訂單 →
② 台灣 IDM 強茂 / 台半 / 朋程 / 富鼎  — 二工認證 + 委外封測/導線架 →
③ 中下游 / 架線 + 代工產能 順德 / 捷敏 / 界霖（+ 漢磊 / 世界先進 / 聯電 代工守門人）

越往下游,基期越低、營運槓桿越大 → 成長彈性越高。EP672 補充:整條鏈的瓶頸是「產能」,故第 ③ 段的代工廠從「配角」升格為「關卡守門人」——沒有六/八吋與 SiC/GaN 產能,二工與外溢都拼不出貨。(★ 漢磊、世界先進為本報告把 EP672 之代工主題對應到的台股標的,股癌本集未逐一點名個股。)

四、成長幅度比較(最新月營收 YoY)

海外 IDM 以 FY2025 全年營收 YoY 表示(景氣低谷,2026 為復甦年);台股為 2026/05 單月營收 YoY。 中下游 / 架線整段營收 YoY 普遍 +18~37%,明顯領先。EP672 的「限速器」意味這條成長曲線更不易過熱、可延續更久。

五、判讀前提:產業循環的兩種烈度(本益比悖論 × 限速器)

估值與「肉」要怎麼讀,先看這個產業屬於哪一種循環——這決定 PE 能不能用、市場 front-run(提前反映)有多猛。

⚠️ 本益比悖論:循環股的 trailing PE 會反向騙人——谷底 EPS 低 / 負 → PE 爆高,常是買點;頂部 EPS 高 → PE 看似便宜,常是賣點。故估值要用 forward PE,並用獨立 through-cycle 研報(晨星公允價值)看穿循環驗證。
🟢 EP672「限速器」的補強:消費/車用疲弱壓住了需求斜率,使這波功率漲價不會一次衝到頂(避免需求破壞與報價崩壞)。對溫和循環而言,這等於把「頂部」往後推、把循環拉長 → front-run 的提前反映幅度被分散,「肉被吃完」的時間軸要拉長看,而非單一復甦年結束。

對本報告海外 IDM 的意涵:功率 / 類比 IDM 屬溫和循環(谷底 ON/STM/TXN 都沒虧過),forward PE 仍可用;晨星 through-cycle 公允價值已把復甦正常化計入,ON 現價仍高於公允價值 +35% → 結構成長大致已 price-in。 EP672 的修正:既然循環更久更長(99+ 交期、需求韌性),海外 IDM 的「貴」更可能是「貴得久」而非「即將反轉」——故定位仍是「先行指標 + 需求韌性確認者」,但成長維度評分微幅上調(見第十節)。

十一、結論與配置思路(EP672 更新)

🥇 肉最多:中下游 / 架線 + 代工產能

成長幅度三段最大(營收 YoY +18~37%)、且伴隨毛利率擴張(順德 12%→19%、界霖 12%→19%)的營運槓桿,完全對應股癌「外溢效應」。 EP672 加碼:「產能即王道」把代工廠從配角拉到關鍵位置(股癌只講「六/八吋代工股已轉強」、未點名個股)——本報告對應到世界先進 (5347)(6/8 吋代工)、漢磊 (3707)(SiC/GaN 代工)為缺貨循環守門人,外資已大買卡位。 估值分歧(已查證重算):捷敏-KY (6525) 前瞻 PE ~21(全組最低)、殖利率 3.5% 為 CP 值最高;順德 (2351) 成長最猛,前瞻 PE 經查證由誤標 71 修正為 ~42(與台半/強茂同級),作為全球功率導線架龍頭(~17%)其實比原先想的有吸引力——但 PEG 0.4 屬循環復甦一次性失真,須留意獲利能否延續。

🥈 居中:台灣 IDM（EP672 受惠「二工認證」上修）

強茂、台半站上波段新高、投信買盤強,前瞻 PE 仍 42–47、PEG 偏高;台半 (5425) PEG ~2.0 相對合理。 EP672 補充:IDM 99+ 交期逼出 second-source,台系高壓 / 伺服器 MOSFET 認證與出貨機會大增 → 訂單動能維度上修。 警訊:富鼎 (8261) 營收近持平、外資大賣、股價卻漲逾年線 109%,追高風險高;朋程營收衰退(EV 競爭、屬 EP672 點名的疲弱消費/車用)。

🥉 肉最少(但循環更久):海外 IDM

2026 將迎景氣復甦(TXN 已 +13% 領先),forward PE 已 38–52 倍、估值歷史高分位,現價高於晨星公允價值(ON +35% 已被評為高估),純價差空間有限。 EP672 修正:Lead Time 99+ 與「限速器」意味循環走更久更長,海外 IDM 的成長韌性比帳面復甦更強——故成長維度評分微上調,角色定位為「先行指標 + 需求韌性確認者」,而非單純賺價差標的。

⚠️ 風險(含 EP672 新增)

  • 需求破壞(EP672):價格漲過頭、消費端撐不住會鎖死需求;留意漲價是否傳導到終端定價。
  • 已大漲:台股這批多站上年線 50~109%、區間位階 90~100%,屬「對的故事 + 不便宜的價格」,追高須留意回檔(EP672 也認中途可能 −30%)。
  • 單股噴出 vs 族群表態(EP672):若非族群同步而是單一大噴,警惕人為操作。
  • 拿不到產能的小 Fabless(EP672):有單也出不了貨,吸引力應調降——反向印證代工產能的稀缺價值。
